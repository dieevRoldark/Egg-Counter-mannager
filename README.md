# Letter-Scanner (Movile first)
Herramienta para contar la cantidad de huevos diarios que salen de la producción agricola.

## Funsiones

Existen 3 producciones de huevos diferenciados por la antiguedad de las gallinas, por ejemplo, galpon 1(las primeras), galpon 2(las segundas), galpon 3(las terceras).

- Contar cuantos huevos salieron de "las primeras"
- Contar cuantos huevos salieron de "las segundas"
- Contar cuantos huevos salieron de "las terceras"

Mostrar el total y el promedio de cada galpon.
